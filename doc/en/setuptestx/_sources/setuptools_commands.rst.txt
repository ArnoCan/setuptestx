Test Automation Commands

.. _SETUP_TESTX:

testx
^^^^^

The *testx* command supports unified and extended tests based on an informal directory structure.
The procedure is in particular designed for the combined test of multiple package sources without 
required installation.
The structure in particular supports simplified distributed bulk-tests within cloud environments.
