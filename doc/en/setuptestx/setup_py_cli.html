5.2. setup.py CLI¶

The setuplib provides the extensions for the standard setup.py of the setuptools / distutils.

5.2.1. SYNOPSIS¶

setup.py [OPTIONS] [COMMANDS]

5.2.2. OPTIONS¶

See:

  • setup.py: [setuptools] and [setuppy]

  • extensions by setuplib: [Global-OPTIONS]

5.2.3. COMMANDS¶

See:

  • setup.py: [setuptools] and [setuppy]

  • extensions by setuplib: [COMMANDS-with-context-OPTIONS]

5.2.4. DESCRIPTION¶

The call interface ‘setuptools’ with extension by ‘setuplib’.

5.2.5. SEE ALSO¶

setup.py, setuplib, [setuptools], [distutils]
